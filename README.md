Проста консольна гра, яка написана на мові C++. Гравець з'являється у випадковій точці на карті, створеній за допомогою масиву.
Гра містить 2 рівні складності: легкий, та складний з відповідно заданою картою, велика або мала.
Сам гравець позначений символом "@", а вихід позначений "X", їх позиція генерується випадково у вільних місцях на карті.
Бар'єр позначений символом "#" та вже заданий за допомогою масиву.

Як грати.
Для початку треба запустити гру обравши рівень складності, під номером 1 легка, під номером 2 важка, для вибору треба ввести в консоль відповідну цифру.
Після запуску гравець побачить поле лабіринту, а також 2 символа, які виділяються кольором, гравець позначений зеленим, а вихід червоним.
Щоб керувати персонажем слід натискати класичні клавіші:
"W" - рух вгору
"S" - рух вниз
"A" - рух ліворуч
"D" - рух праворуч
Рухаючи гравцем треба дійти до виходу - червона літера "X"
Для виходу в меню треба натиснути клавішу "Esc"
